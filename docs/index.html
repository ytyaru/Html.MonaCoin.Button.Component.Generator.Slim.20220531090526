投げモナボタンジェネレータ（WebComponent版）＋アニメ＋コード減量

あなた宛の投げモナボタンを作ります。画像をどれかひとつにすることで減量します。

使い方
  1. mpurseで口座を作る
  2. ブラウザの拡張機能でmpurseをクリックしてパスワードを入れて承認します
  3. あなたのアドレスが出力される
  4. コピーボタンを押す
  5. あなたのWEBサイトに貼り付ける
  6. あなたのWEBサイトに、あなた宛の投げモナボタンが表示される

もしアドレスが出なければmpurseを承認後、リロードやアドレスを取得するボタンを押してみてね。手で入力してもOK。
